\chapter{引言}

\section{宇宙学分析的方法与手段}

现代宇宙学的一个核心目标，是利用观测数据精确测量宇宙学参数，并由此检验宇宙的组成、几何结构和演化历史。不同观测探针从不同角度提供信息：星系巡天项目（如 DESI、SDSS）通过测量星系光谱，重建星系在三维空间中的分布，并利用星系成团性约束宇宙学模型\citep{adame2025desi}；引力透镜观测则利用宇宙微波背景辐射的透镜畸变（CMB 透镜，如 Planck 卫星项目）或星系形态在引力场中的微弱畸变（弱引力透镜，如 Euclid、CSST 项目），反推出宇宙中的物质分布\citep{planck2016cosmological,euclid2020forecast,kitching2015weaklensing}；超新星作为标准烛光，其光变曲线和亮度距离关系也可用于限制宇宙膨胀历史。

这些探针虽然观测对象和数据形式不同，但在参数推断中通常遵循相近的贝叶斯分析框架。具体而言，观测数据首先被压缩为功率谱、关联函数、距离模数等数据向量；随后，在给定宇宙学模型和模型参数的条件下，计算相应的理论预期以及数据向量的误差分布。通过比较观测数据与理论预期，并在参数空间中采样，即可得到宇宙学参数的后验概率分布。在这一过程中，协方差矩阵决定了不同数据分量的误差大小及其相关性，是连接观测数据与参数约束的重要桥梁。

\subsection{协方差矩阵与 N 体模拟}

协方差矩阵描述数据向量在重复观测之间的涨落，通常包含统计误差和系统误差两类来源。对于理想化情形，协方差矩阵有时可以通过解析方法计算；但在真实巡天中，观测几何、选择函数、非线性引力演化以及示踪天体形成机制都会显著增加问题复杂度，使得解析表达式难以准确获得。因此，在星系巡天、弱引力透镜等低红移探针的分析中，协方差矩阵往往需要依赖大量数值模拟进行估计。

目前，宇宙学模拟中最常用也最可靠的基准方法是 N 体模拟。在冷暗物质模型假设下，N 体模拟先根据给定宇宙学参数生成初始时刻的暗物质粒子分布，再通过粒子之间的引力相互作用演化系统，最终得到目标红移处的暗物质密度分布和速度场\citep{springel2021gadget4}。对于星系巡天、弱引力透镜等具体探针，还需要在暗物质场的基础上进一步构造相应的示踪天体分布。

即使宇宙学参数完全相同，不同模拟实现也会因为初始随机种子不同而产生不同的粒子初始位置。这些初始涨落经过非线性引力演化后，会导致最终的暗物质密度场、速度场以及示踪天体分布产生差异，并进一步表现为数据向量的统计涨落。因此，使用一组相互独立的模拟实现估计数据向量的样本协方差，是宇宙学参数推断中的常见做法。至于仪器标定、观测选择效应等系统误差，也可以在部分情况下通过模拟流程建模，但这并非本文的讨论重点。

\section{宇宙学快速模拟方法}

高精度 N 体模拟能够较准确地刻画暗物质结构的非线性演化，但其计算代价也十分高昂。为了达到现代宇宙学测量所需的体积和分辨率，单个模拟往往需要 Gpc 尺度为边长的模拟盒子以及数亿乃至数十亿个粒子。如此大规模的粒子体系在相互引力作用下演化，通常需要数天到数月的计算时间；而协方差矩阵估计又往往需要上千个相互独立的模拟实现。由此可见，仅依赖完整 N 体模拟难以满足大规模误差估计的需求，发展计算效率更高的宇宙学快速模拟方法具有重要意义。

\subsection{宇宙学快速模拟方法的需求}

随着观测设备不断迭代，新一代巡天项目对快速模拟提出了更高要求。首先，巡天体积持续增大，模拟需要覆盖更大的宇宙空间并处理更多暗物质粒子；其次，多探针联合分析正在成为提高宇宙学参数约束能力的重要方向，而不同探针之间的相关性只有在同一组模拟实现中才能被一致地估计。因此，面向未来巡天的快速模拟方法不能只服务于单一统计量，而应尽可能同时满足多类探针的需求。

具体而言，理想的快速模拟方法应具备以下特征：
\begin{itemize}
  \item 生成速度快，能够在短时间内产生大量用于误差估计的模拟实现；
  \item 可扩展性强，在模拟体积增大或粒子数增加时仍能保持较高效率；
  \item 能够同时提供可靠的暗物质密度场和速度场，在大尺度上保持与 N 体模拟一致，并尽可能恢复小尺度非线性结构。
\end{itemize}

其中，星系巡天通常更关注大尺度成团性信息，而 CMB 透镜和弱引力透镜对小尺度物质分布也较为敏感；速度场则与红移空间畸变、运动学 SZ 效应等探针密切相关。因此，能够同时生成密度场和速度场的快速模拟方法，更适合服务于多探针联合分析。

\subsection{宇宙学快速模拟方法的研究现状与不足}

传统快速模拟方法大多从拉格朗日微扰论出发。拉格朗日微扰论从粒子初始位置出发，通过微扰展开近似求解粒子在引力演化中的位移。例如，DESI DR1 中使用的 EZMock 方法基于 Zel'dovich 近似（ZA，即一阶拉格朗日微扰论）；而 DESI DR2 计划采用的模拟则基于增强拉格朗日微扰论（ALPT），该方法建立在二阶拉格朗日微扰论（2LPT）的基础之上\citep{kitaura2013alpt,kitaura2014patchy,chuang2015ezmocks}。

微扰论的优点在于物理图像清晰、实现简单且计算速度较快。在大尺度上，由于密度涨落较小，引力演化接近线性，因此该方法能较好地保留大尺度结构信息。然而在引力演化后期，小尺度结构已进入强非线性区域，微扰展开难以准确描述诸如团块形成、丝状结构收缩以及空洞边界演化等过程。此时，近似轨迹容易发生“壳层穿越”现象，即不同粒子的预测轨迹相互交叉，而方法本身无法进一步描述维里化与绕转运动，导致小尺度结构被过度平滑。尽管结合偏置模型或经验修正后，这类方法能够满足部分当前星系巡天分析的需求，但面对下一代探针对小尺度非线性结构的精度要求，其能力仍然不足。

除拉格朗日微扰论外，也可在欧拉空间中直接研究固定位置上的密度场与速度场演化\citep{jeong2006perturbation,taruya2018grid}。然而，将欧拉空间微扰理论直接应用于快速模拟时，强非线性区域中的微扰条件会被破坏，甚至出现负密度等非物理结果。基于有效场论的暗物质或星系分布模拟能够在一定程度上吸收小尺度物理的影响，但通常需要拟合额外的自由参数，且计算效率仍难以满足大批量协方差估计的需求\citep{carrasco2014eft}。

近年来，机器学习方法为宇宙学快速模拟提供了新的方向。随着神经网络和生成模型的发展，已有研究尝试使用机器学习从近似初始条件或低精度结果中生成更接近 N 体模拟的暗物质场\citep{he2019learning,oliveira2020fast,jamieson2023field,jamieson2024field}。其中，U 型网络能够同时利用全局上下文和局部细节，在三维场生成任务中具有较强表达能力。已有工作尝试使用 U 型网络架构预测任意红移下暗物质粒子的位移场和动量场，TUNeS 也采用类似思路预测任意红移下的物质密度场\citep{kang2026tunes}。不过，这些方法通常面对较高粒子分辨率下的复杂学习任务，训练难度较大；同时，许多方法未能同时给出可靠的粒子速度场，而速度场正是红移空间畸变、运动学 SZ 效应等探针模拟中不可或缺的物理量。

\subsection{本文研究内容与贡献}

针对上述问题，本文提出一种基于尺度分离思想的宇宙学快速模拟方法。该方法首先利用 ZA 快速生成位移场和速度场的大尺度基准；随后，通过两个独立的 WGAN-GP 残差网络分别学习 N 体模拟相对 ZA 的位移残差和速度残差，以补全非线性引力演化带来的小尺度结构细节。换言之，ZA 负责保留可靠的大尺度模式，神经网络则主要学习小尺度非线性修正，从而在降低学习难度的同时提高生成结果的物理一致性。

本文工作的主要贡献如下：
\begin{itemize}
  \item 同时生成暗物质密度场和速度场：位移场可进一步构造密度场并服务于弱引力透镜等探针，速度场可服务于红移空间畸变、运动学 SZ 效应等探针，两者结合有助于多探针联合分析；
  \item 采用残差学习策略，将大尺度线性结构交由 ZA 描述，使神经网络专注于学习小尺度非线性修正，降低了训练任务复杂度；
  \item 结合分块推理、高通滤波和旋转平均等策略，使模型在保持快速生成优势的同时，能够自然地扩展到大体积的模拟；
  \item 从单点统计、两点统计（功率谱）、三点统计（双谱）和协方差矩阵等多个角度验证方法效果，系统评估其相对于传统微扰论快速模拟的改进。
\end{itemize}
